# Gym Management System (DBMS Project)

This project is a Database Management System (DBMS) mini project based on a Gym Management System.

## 📌 Features
- Manage gym members
- Trainer details
- Membership plans
- Payment tracking

## 🛠️ Technologies Used
- SQL (MariaDB)
- DBMS Concepts

## 📂 Database Tables
- Member
- Trainer
- Plan
- Membership
- Payment

## 📊 Operations Performed
- DDL (CREATE TABLE)
- DML (INSERT, UPDATE, DELETE)
- DQL (SELECT queries, JOIN, GROUP BY)
- View creation
- Indexing

## 📎 Files Included
- SQL Script
- Project Report (PDF)
- ER Diagram
